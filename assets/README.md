# Assets Folder

Place the following image files in this folder:

1. **icon.png** (1024x1024) - App icon for app stores
2. **splash.png** (1284x2778) - Splash screen background
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
4. **favicon.png** (48x48) - Web favicon

## Quick Start
For development, you can use any placeholder images with the correct dimensions.

## Recommended Tools
- Use [Figma](https://figma.com) or [Canva](https://canva.com) to create these assets
- Use the TaxClarity green (#0A8A3E) as the primary brand color
